```console

shikhar@MacBook~ $ cd YukkiMusicBot
shikhar@MacBook~ $ pip3 install -U -r requirements.txt
shikhar@MacBook~ $ cp sample.env .env
```

> Edit .env with your values and then start bot with
```console
shikhar@MacBook~ $ bash start
```
